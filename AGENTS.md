# AGENTS.md

## Scope

These instructions apply to the repository root at `/Users/julian/Code/cli-microsoft365`.

## Environment

- Use Node.js `24.13.1` from `.tool-versions`.
- Install dependencies with `npm ci`.
- Build the project with `npm run build`.

## Validation

- Run `npm run lint` for TypeScript linting.
- Run `npm run test:test` for the Mocha test suite.
- Run `npm test` before shipping broad changes because it includes the version check, lint, and coverage run.

## Repository Layout

- `src/` contains the CLI source code.
- `src/m365/**/commands/` contains workload-specific command implementations.
- `scripts/` contains build and maintenance scripts used by `npm run build` and test flows.
- `docs/` contains the Docusaurus documentation site and should only be touched for docs work.

## Editing Guidance

- Follow existing command and test patterns in neighboring files before introducing new structure.
- Prefer focused changes and add or update tests when command behavior changes.
- `dist/` is generated by the build and should not be edited by hand.
